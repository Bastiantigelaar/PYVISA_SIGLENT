# Basic GUI with SIGLENT FUNCTION GENERATER

based on the programming guide of siglent and tektronix.


# supported devices:
## scopes:
TDS200,TDS1000, TDS2000, and TPS2000 Series Digital Oscilloscopes
## function generators: 
all siglent generators

first install:
```bash
pip install pyvisa
```
## second install
second install:
```bash
pip install numpy
```
